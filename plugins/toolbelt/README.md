# toolbelt

My collection of day-to-day tools to get the job done.

## Installation

Register the marketplace (one-time):

```text
claude plugin marketplace add glnds/claude-marketplace
```

Install the plugin:

```text
claude plugin install toolbelt@claude-marketplace
```
